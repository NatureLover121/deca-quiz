Using this Site

There are several quizzes on this website that can help you to find the DECA event most suited to you. We advise that you begin with the "Event Category" quiz, and then proceed to the "Prepared Events," "Roleplays" quizzes as needed. You should end with the "Clusters" quiz.

Based on your results, you will be able to filter the vast list of 60 events on deca.org/compete down to a much more manageable list of events to choose from.
